Merge Sale and Purchase Order Lines

Sometimes in Odoo user may add similar products multiple times in the orderline for bulk sale/purchase order, resulting increase the number of orderlines. This odoo app helps user to merge or combine sale and purchase order lines of similar product and quantity and remove the duplicate records. User can configure whether to merge the lines automatically or manually. it is user for to manage SO PO line in Odoo EPR Module.

Merge Sale and Purchase Order Lines

User can merge sale and purchase order lines of similar product, and product quantity will be merged.

Auto and Manual Merge Options

User have options to merge sale and purchase order lines with manual and auto options.

Unlink Old Order Lines After Merge

After merge sale and purchase order, lines will be removed from old order and merge quantity in new order.

Description

Manual Merge Sale Order Lines


Merge Menu

Under Sale Order > Action menu user can see "Merge Sale Lines" menu.



User click on "Merge Sale Lines" menu a new wizard will open to confirm merge operation.



All same product line on sales order will merge with ordered quantity.


Auto Merge Sale Order Lines


Configuration

Under Sales > Configuration > Settings menu enable "Auto Merge Sale Order Lines".



User create a new order or edit an existing order all same product and quantity will be merged.


Manual Merge Purchase Order Lines


Merge Menu

Under Purchase Order > Action menu user can see "Merge Purchase Lines" menu.



User click on "Merge Purchase Lines" menu a new wizard will open to confirm merge operation.



All same product line on purchase order will merge with ordered quantity.


Auto Merge Purchase Order Lines


Configuration

Under Purchase > Configuration > Settings menu enable "Auto Merge Purchase Order Lines".



User create a new order or edit an existing order all same product and quantity will be merged.
